```bash

```

## 2. Create a virtual environment (recommended)

**Using Python venv:**

```bash

```

## 3. Install dependencies

```bash
pip install -r requirements.txt
```

This installs the necessary libraries for raster processing, vector handling, and classification:

* rasterio
* geopandas
* shapely
* numpy
* pandas
* scikit-learn

### **Expected Inputs**

6. rededge
7. nir
8. gndvi
9. lci
10. ndre
11. ndvi
12. osavi
13. dsm

#### **2. `data/training/training_points.gpkg`**
